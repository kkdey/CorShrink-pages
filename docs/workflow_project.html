Codes to reproduce analysis from Dey and Stephens (2018+)

Here are the analysis RmD files for reproducing the figures from our paper. To run these scripts, first clone the source directory

git clone https://github.com/kkdey/CorShrink-pages.git

Then go to the analysis directory

cd CorShrink-pages/analysis

The paths of the data files in the scripts below (under Figures and Supplementary Figures) are relative to this directory.
