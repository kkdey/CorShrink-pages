Simulation studies partial correlation

Kushal K Dey

5/22/2018

  • Packages
  • Hub correlation
  • Toeplitz matrix

We perform box plot representation of the Frobenius distance between the population correlation matrix and correlation matrix estimators - CorShrink, Shafer Strimmer and the GLASSO methods.

Packages

library(gridExtra)
library(ggplot2)
library(scales)

Hub correlation

We simulate data from a hub structured correlation matrix under different settings of \(n\) and \(p\).

First run the code in

source("../code/Figure5/corshrink_hub_pcor.R")  ## n=50, p=100 

## last line : save(frob_vals, file = paste0("hub_cmd_boot_n_", n, "_P_", P, "_results_pcor.rda"))
## change the file to the destination and file name you prefer.

Change the \(n\) and \(p\) accordingly, we pick \(p=100\) and vary \(n\) from \(30, 50, 100, 1000\).

The final output from these files were saved under shared_output/Pcor/ folders and can be read as follows.

hub_n_10_P_100 <- get(load("../shared_output/Pcor/hub_cmd_boot_n_30_P_100_results_pcor.rda"))
hub_n_50_P_100 <- get(load("../shared_output/Pcor/hub_cmd_boot_n_50_P_100_results_pcor.rda"))
hub_n_100_P_100 <- get(load("../shared_output/Pcor/hub_cmd_boot_n_100_P_100_results_pcor.rda"))
hub_n_1000_P_100 <- get(load("../shared_output/Pcor/hub_cmd_boot_n_1000_P_100_results_pcor.rda"))

df <- data.frame("method" = factor(c(rep("empirical", 50), rep("corpcor", 50), rep("corshrink", 50),
                              rep("pdsce",50), rep("glasso:1e-02", 50),
                              rep("glasso:0.1", 50), rep("glasso:0.5", 50), rep("glasso:1", 50)), 
                              levels = c("empirical", "corpcor",
                                        "corshrink", "pdsce",
                                        "glasso:1e-02",
                                        "glasso:0.1",
                                        "glasso:0.5",
                                        "glasso:1")),
                 "distance" = log(c(hub_n_10_P_100[,1], hub_n_10_P_100[,2],
                                    hub_n_10_P_100[,3], hub_n_10_P_100[,4],
                                    hub_n_10_P_100[,5], hub_n_10_P_100[,6],
                                    hub_n_10_P_100[,7], hub_n_10_P_100[,8])))

p <- ggplot(df, aes(method, distance, color = method)) + ylab("log(distance)")
p1 <- p + geom_boxplot() + theme_bw() + theme(legend.position="none") +
  theme(axis.title.x=element_blank(),
        axis.text.x=element_blank(),
        axis.ticks.x=element_blank()) + ggtitle(paste0("n = ", 30, " p = ", 100)) +
scale_y_continuous(breaks= pretty_breaks())
